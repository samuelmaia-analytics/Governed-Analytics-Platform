# Power BI


## Acesso Rápido

- Repositório: `https://github.com/samuelmaia-analytics/SAMUEL_MAIA_DDF_TECH_032026`

- Ativo principal na Dadosfera: `https://metabase-treinamentos.dadosfera.ai/model/2719-fact-orders-dashboard`

Esta pasta concentra os artefatos finais do bônus de BI externo, alinhados ao formato de entrega do case e à narrativa de consumo analítico complementar ao Streamlit.

## Papel do bônus

O material de Power BI não substitui o dashboard principal do projeto. Ele amplia a solução para um formato de BI mais tradicional, útil para exploração gerencial, drill-down e demonstração de multiconsumo da mesma camada analítica.
